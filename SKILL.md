### 第二步：处理图片

使用图片CDN代理（images.weserv.nl）构建处理URL：

#### URL构建规则

基本格式：
```
原始URL: https://example.com/image.jpg
处理后:  https://images.weserv.nl/?url=example.com/image.jpg&w=300&h=400
```

#### 参数说明

| 参数 | 说明 | 示例 |
|:----:|------|------|
| `url` | 图片URL（去掉https://） | `example.com/image.jpg` |
| `w` | 宽度（px） | `w=300` |
| `h` | 高度（px） | `h=400` |
| `fit` | 裁剪模式：cover/contain/inside/outside | `fit=cover` |
| `we` | 水印URL | `we=watermark.com/logo.png` |
| `wea` | 水印位置：center/top/bottom/left/right 组合 | `wea=bottom-right` |
| `web` | 水印透明度（0-1） | `web=0.5` |
| `f` | 输出格式：jpg/png/webp/avif | `f=webp` |
| `q` | 质量（1-100） | `q=80` |
| `b"|`"$` | 模糊半径 | `blur=5` |
| `t` | 锐化强度 | `t=3` |
| `filt` | 滤镜：grayscale/brightness/contrast | `filt=grayscale` |
| `fn` | 亮度（-255~255） | `fn=50` |
| `fc` | 对比度（-255~255） | `fc=30` |

#### 常见处理示例

```
1. 缩放至300px宽，自动高度
   https://images.weserv.nl/?url=example.com/image.jpg&w=300

2. 缩放至300x400并裁剪
   https://images.weserv.nl/?url=example.com/image.jpg&w=300&h=400&fit=cover

3. 转为WebP，质量80
   https://images.weserv.nl/?url=example.com/image.jpg&f=webp&q=80

4. 添加右下角水印
   https://images.weserv.nl/?url=example.com/image.jpg&we=watermark.com/logo.png&wea=bottom-right&web=0.5

5. 灰度+亮化
   https://images.weserv.nl/?url=example.com/image.jpg&filt=grayscale&fn=20
```

## 特殊场景处理

### 场景1：图片URL无效或无法访问
- 检查URL格式是否正确
- 建议用户提供可公开访问的图片URL
- 不支持本地文件或需要登录的图片

### 场景2：处理URL过长
- 建议用户分步处理（先缩放→再格式转换）
- 使用短链接服务（如需要）

### 场景3：需要文字水印
- CDN代理不直接支持文字水印
- 建议使用图片水印替代（先制作文字水印图片）
- 或推荐在线文字水印工具和操作步骤

### 场景4：需要更高级的编辑
- 照片合成、抠图等不在本工具范围内
- 推荐使用专业图片编辑工具
- 提供AI图像处理工具的建议

### 场景5：用户未提供图片URL
- 引导用户提供图片URL
- 告知支持的图片格式：JPG/PNG/GIF/WebP/AVIF
- 告知图片大小限制：建议不超过20MB

### 场景6：批量处理不同参数
- 建议分批处理（每批统一参数）
- 或逐条指定处理参数
